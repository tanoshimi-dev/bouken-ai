# Claude Codeとは

## はじめに

Claude Codeは、Anthropic社が開発したAIコーディングアシスタントです。ターミナル上で直接動作し、開発者がコードの作成、編集、デバッグ、そしてプロジェクト管理を効率的に行えるように設計されています。

通常のClaude（チャットインターフェース）とは異なり、Claude Codeはあなたの開発環境に直接統合され、ファイルシステムへのアクセス、コマンドの実行、そしてプロジェクト全体のコンテキストを理解した上でのサポートが可能です。

## Claude Codeの主な特徴

### 1. ターミナルネイティブ

Claude Codeはターミナル（コマンドライン）から直接起動し、操作します。IDEやブラウザを切り替える必要はありません。

```bash
# Claude Codeの起動
claude

# 質問を直接渡して起動
claude "このプロジェクトの構造を説明して"
```

### 2. ファイルシステムへの直接アクセス

Claude Codeはプロジェクト内のファイルを直接読み書きできます。これにより、コードの変更提案だけでなく、実際にファイルを編集することが可能です。

```bash
# プロジェクトディレクトリで起動すると、そのプロジェクトのファイルにアクセス可能
cd my-project
claude
```

### 3. コマンド実行能力

テストの実行、ビルド、Gitの操作など、シェルコマンドを直接実行できます。

```bash
# Claude Code内でコマンド実行を依頼
> テストを実行して結果を確認して
> このプロジェクトをビルドして
> 最新のgitログを見せて
```

## 通常のClaudeとの違い

| 機能 | 通常のClaude | Claude Code |
|------|-------------|-------------|
| 動作環境 | ブラウザ/API | ターミナル |
| ファイル操作 | 不可 | 可能 |
| コマンド実行 | 不可 | 可能 |
| プロジェクト理解 | 限定的 | 包括的 |
| コード編集 | 提案のみ | 直接編集可能 |
| Git操作 | 不可 | 可能 |

## 基本的な使い方

### 対話モードでの起動

```bash
# 最もシンプルな起動方法
claude
```

起動後、自然言語で指示を出すことができます。

### ワンショットモード

```bash
# 1つの質問を投げて結果を得る
claude "package.jsonの依存関係を一覧にして"
```

### パイプを使った入力

```bash
# コマンドの出力をClaude Codeに渡す
cat error.log | claude "このエラーの原因を分析して"

# diffの内容をレビューしてもらう
git diff | claude "この変更をレビューして"
```

## Claude Codeが得意なこと

- **コード生成**: 新しい機能やコンポーネントの作成
- **バグ修正**: エラーの原因特定と修正
- **リファクタリング**: コードの品質改善
- **テスト作成**: ユニットテストや統合テストの生成
- **ドキュメント作成**: コードのドキュメンテーション
- **コードレビュー**: 変更内容のレビューとフィードバック

## まとめ

Claude Codeは、開発者のワークフローに自然に溶け込むAIアシスタントです。ターミナルから直接操作でき、プロジェクトのコンテキストを理解した上で、実践的なコーディング支援を提供します。次のレッスンでは、環境のセットアップ方法を学びます。
